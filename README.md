# 📊 RetailIQ Pro  
### Business Intelligence Dashboard • Streamlit • Pandas • Plotly

RetailIQ Pro is an interactive Business Intelligence dashboard designed to analyze retail performance, customer behavior, and product trends using modern Python data tools. It includes dynamic visualizations, KPI tracking, demographic insights, and a comparison module against the popular SuperStore dataset.

This project is ideal for showcasing data analytics, dashboard development, and Python engineering skills.

---

## 🚀 Features

### **RetailIQ Dataset**
- Revenue, customer, transaction, and unit KPIs  
- Category‑level revenue analysis  
- Time‑series revenue trends  
- Customer demographics (age, gender)  
- Customer spending behavior  
- AI‑style insights generated from the dataset  
- Sidebar filters for category and gender  
- CSV export functionality  

### **SuperStore Dataset**
- Sales, profit, and quantity KPIs  
- Profit by region  
- Sales by category  
- Discount vs profit scatter analysis  
- Region × category heatmap  
- CSV export functionality  

### **Comparison Dashboard**
- RetailIQ vs SuperStore revenue comparison  
- Units sold comparison  
- Profit benchmarking  
- Visual comparison charts  

---

## 🧠 Smart Insights

RetailIQ Pro includes automated insights such as:
- Top‑performing product category  
- Average customer age  
- Highest revenue day  
- Spending patterns by age  

These insights help users quickly understand trends without manually digging through the data.
